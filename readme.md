# Projeto de Aplicação Web Api

Exemplo de projeto web api

### Pré-requisitos

Antes de rodar o projeto é necessário executar o seguinte comando:

```
dotnet ef database update
```
Comando criará um banco de dados local "LocalDb"


### Após criar o banco de dados é só rodar o projeto. Abrirá uma página Swaggwer com todas as apis da entidade Cliente
